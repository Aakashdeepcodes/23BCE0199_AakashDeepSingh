QUESTION 8: Remove the 'scholarship' field from all CS students
            using $unset.
================================================================

QUERY:
------
db.students.updateMany(
  { dept: "CS" },
  { $unset: { scholarship: "" } }
)

EXPLANATION:
  • updateMany          → applies the update to ALL matching documents
  • { dept: "CS" }      → filter: only CS department students
  • $unset: { scholarship: "" } → removes the 'scholarship' field
    completely from each matched document (the value "" is ignored;
    any value works with $unset)

AFFECTED STUDENTS (dept: "CS"):
  _id 1 – Ananya,  _id 3 – Meena,  _id 6 – Dinesh

EXPECTED OUTPUT:
  { acknowledged: true, matchedCount: 3, modifiedCount: 3 }

QUESTION 14: Find departments whose average salary
             is greater than 50000.
================================================================

QUERY:
------
db.employees.aggregate([
  {
    $group: {
      _id:       "$department",
      avgSalary: { $avg: "$salary" }
    }
  },
  {
    $match: {
      avgSalary: { $gt: 50000 }
    }
  },
  {
    $project: {
      _id:        0,
      department: "$_id",
      avgSalary:  { $round: ["$avgSalary", 2] }
    }
  },
  {
    $sort: { avgSalary: -1 }
  }
])

EXPLANATION (pipeline stages):
  Stage 1 – $group   : group employees by department, calculate
                       average salary per group using $avg
  Stage 2 – $match   : filter groups where avgSalary > 50000
  Stage 3 – $project : rename _id to department; round avgSalary
                       to 2 decimal places
  Stage 4 – $sort    : order results by avgSalary descending

AVERAGE SALARY BY DEPARTMENT:
  IT        → (45000 + 75000 + 68000) / 3 = 62666.67  ✔ > 50000
  HR        → 55000 / 1                  = 55000.00  ✔ > 50000
  Finance   → 50000 / 1                  = 50000.00  ✗ NOT > 50000
  Marketing → 42000 / 1                  = 42000.00  ✗ NOT > 50000

EXPECTED OUTPUT:
  { department: "IT", avgSalary: 62666.67 }
  { department: "HR", avgSalary: 55000.00 }

QUESTION 15: Find minimum salary in each department.
================================================================

QUERY:
------
db.employees.aggregate([
  {
    $group: {
      _id:    "$department",
      minSalary: { $min: "$salary" }
    }
  },
  {
    $project: {
      _id:        0,
      department: "$_id",
      minSalary:  1
    }
  },
  {
    $sort: { minSalary: 1 }
  }
])

EXPLANATION (pipeline stages):
  Stage 1 – $group   : group employees by department, find the
                       minimum salary in each group using $min
  Stage 2 – $project : rename _id to department for clarity
  Stage 3 – $sort    : order results by minSalary ascending

MIN SALARY BY DEPARTMENT (calculated from dataset):

QUESTION 16: Rename field 'city' to 'location' in employees.
================================================================

QUERY:
------
db.employees.updateMany(
  {},
  { $rename: { "city": "location" } }
)

EXPLANATION:
  • updateMany({}, ...)    → empty filter {} matches ALL documents
  • $rename: { "city": "location" } → renames the field 'city'
    to 'location' across every employee document
  • If a document does not have the 'city' field, $rename
    simply ignores it (no error thrown)

BEFORE (example):

AFTER (example):
  { empId: 101, name: "Arun", location: "Chennai", ... }

EXPECTED OUTPUT:
